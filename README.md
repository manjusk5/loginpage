# Netflix Login Page

Full-stack Netflix-style login with React (Vite) + Express.

## Quick start

```bash
# Install root helper (optional, for running both at once)
npm install

# Terminal 1 – backend
cd server
npm start

# Terminal 2 – frontend
cd client
npm run dev
```

Open http://localhost:5173

## Features

- Netflix-inspired responsive login UI (Tailwind CSS)
- Frontend validation (empty fields)
- Express API with mock user authentication
- Error messages for invalid login
- Redirect to dashboard on success
